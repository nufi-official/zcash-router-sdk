# @zcash-router-sdk/zcash-account-mnemonic

Zcash account management library with mnemonic support for the Zcash Router SDK.

## Features

- ✅ **TypeScript** - Full TypeScript support with type declarations
- ✅ **Dual Format** - Supports both ESM and CommonJS
- ✅ **Browser Compatible** - Works in Node.js and browsers
- ✅ **Tree-shakeable** - Optimized for bundle size
- ✅ **Fully Tested** - Comprehensive test coverage with Vitest

## Installation

```bash
npm install @zcash-router-sdk/zcash-account-mnemonic
```

```bash
yarn add @zcash-router-sdk/zcash-account-mnemonic
```

```bash
pnpm add @zcash-router-sdk/zcash-account-mnemonic
```

## Development

### Prerequisites

- Node.js 18+
- pnpm (recommended) or npm/yarn

### Setup

```bash
# Install dependencies (from monorepo root)
pnpm install

# Run tests
pnpm test

# Run tests in watch mode
pnpm test:watch

# Build the library
pnpm build

# Development mode (watch)
pnpm dev

# Lint code
pnpm lint

# Format code
pnpm format
```
